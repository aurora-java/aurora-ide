Aurora Project

Aurora工程是在普通Java Web工程的基础之上增加aurora nature而来的。 具体的做法是：在工程上点右键，设置为Aurora工程。再次操作则取消设置。 Aurora工程需要具备以下特点:

  • 具有WEB-INF目录
  • WEB-INF目录下有classes、lib目录
  • Aurora工程所必需的配置文件 
      [webRoot]
        └─WEB-INF
            ├─ [各种其他配置文件]
            ├─classes
            ├─lib
            │  ├─ [其他jar包]
            │  └─ aurora.jar
            └─ web.xml

Aurora工程会有一个额外的工程属性配置页面，可以通过右击工程 > Properties (Alt + Enter)（或者，选定工程，菜单Project > Properties），进入Aurora工程设置。 新建Aurora工程，或设置为Aurora工程时，这些工程属性会自动设置，用户也可以手动更改设置。
注意：当一个工程有多个WEB-INF目录时，请特别注意一下此项设置。

正确设置Aurora工程属性对于IDE有重要意义，IDE会据此获取web主目录、bm主目录 、数据库连接配置等重要信息。
此外，Aurora原型工程必须关联一个配置正确的Aurora工程。
